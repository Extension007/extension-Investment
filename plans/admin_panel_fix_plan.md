# План исправления проблемы с отображением админ-панели

## Проблема
Интерфейс админ-панели "появляется на долю секунды и пропадает". Это указывает на возможную JavaScript ошибку, которая приводит к исчезновению контента.

## Анализ
После анализа кода выявлены следующие возможные причины:

1. На странице админ-панели подключаются общие скрипты, которые предназначены для других страниц и могут вызывать ошибки на странице админ-панели.
2. В общем скрипте `public/script.js` есть проверка `window.IS_CABINET_PAGE`, которая может приводить к пропуску инициализации необходимых функций на странице админ-панели.
3. В JavaScript коде админ-панели (внутри шаблона `views/admin.ejs`) могут быть вызовы функций, которые не существуют или вызывают ошибки.

## Решение
Необходимо внести следующие изменения:

1. Упростить JavaScript в шаблоне админ-панели, убрав дублирующийся код и потенциально проблемные вызовы
2. Убедиться, что все необходимые функции инициализируются корректно
3. Обеспечить правильную загрузку YouTube IFrame API
4. Убрать вызовы функций, которые могут не существовать в контексте админ-панели

## Конкретные изменения

### 1. В шаблоне `views/admin.ejs`:
- Убрать дублирующийся код, связанный с видео-плеерами, так как они не используются на админ-панели
- Упростить обработчики событий, оставив только необходимые для админ-панели
- Убедиться, что YouTube IFrame API загружается корректно

### 2. Оставить только необходимую функциональность:
- Обработка форм создания товаров/баннеров
- Функции модерации (одобрение, отклонение)
- Блокировка/разблокировка карточек
- Обработка выхода из системы

## Тестирование
После внесения изменений необходимо протестировать:
1. Отображение админ-панели
2. Работу всех функций админ-панели
3. Корректность работы форм
4. Отсутствие JavaScript ошибок в консоли
